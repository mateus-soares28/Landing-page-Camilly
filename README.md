# Camily Rumpff - Landing Page

Landing page institucional para a Camily Rumpff Estetica Corporal, com foco em apresentacao de tratamentos, resultados e formulario de contato via WhatsApp.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- GSAP para animacoes

## Estrutura

```text
.
├── index.html
├── css/
│   ├── reset.css
│   ├── style.css
│   └── responsive.css
├── js/
│   └── script.js
└── assets/
    └── img/
```

## Como Rodar

Abra o `index.html` no navegador ou use um servidor local:

```bash
python -m http.server 4173
```

Acesse:

```text
http://127.0.0.1:4173
```

## Status

Projeto responsivo para desktop, MacBook, tablet e celular, pronto para publicacao.
